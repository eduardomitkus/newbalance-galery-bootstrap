# New Balance Gallery - Bootstrap

Museu virtual sobre a trajetória da New Balance no futebol mundial.

## Estrutura

- `index.html` - Sala 1: Origens da NB
- `evolution.html` - Sala 2: Evolução Estética
- `clubs.html` - Sala 3: Clubes Atuais

## Como rodar localmente

### Opção 1: Python (recomendado)
```bash
python3 serve.py
```
O servidor abrirá automaticamente em `http://localhost:8000`

### Opção 2: Python simples
```bash
python3 -m http.server 8000
```
Depois acesse `http://localhost:8000` no navegador

### Opção 3: Live Server (VS Code)
Se você usa VS Code, instale a extensão "Live Server" e clique com botão direito no `index.html` → "Open with Live Server"

## Tecnologias

- HTML5
- CSS3
- Bootstrap 5.3.2 (via CDN)
